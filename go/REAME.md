# String formatting map
|                | Abbr         |
| -------------- | ------------ |
| Type           | %T           |
| Bool           | %t           |
| Int            | %d           |
| Binary         | %b           |
| Character      | %c           |
| Heximal        | %x           |
| Float          | %f / %e / %E |
| String         | %s / %q      |
| Pointer        | %p           |
| Number & Width | %6d          |
| Float & Width  | %6.3f        |
| String & Width | %10s / %-10s |

# Golang Keywords
| -        | -           | -      | -         | -      |
| -------- | ----------- | ------ | --------- | ------ |
| break    | default     | func   | interface | select |
| case     | defer       | go     | map       | struct |
| chan     | else        | goto   | package   | switch |
| const    | fallthrough | if     | range     | type   |
| continue | for         | import | return    | var    |
